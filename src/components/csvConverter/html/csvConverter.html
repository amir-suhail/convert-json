Convert to CSV

Here we use ng-csv plugin to convert to a CSV file.

Steps to follow:

  1. Download ng-csv.min.js and angular-sanitize.min.js and add to your project.
  2. Add couple of script tags to your index.html which points to the ng-csv.min.js and angular-sanitize.min.js.
    <script scr="basePath/ng-csv.min.js"></script> <script scr="basePath/angular-sanitize.min.js"></script>
  3. Add ngCsv and ngSanitize as dependencies in your module.
    var myapp = angular.module('myapp', ['ngSanitize', 'ngCsv'])
  4. Add ng-csv directive to the wanted element.
    <button type="button" ng-csv="getArray" csv-header="['Field A', 'Field B', 'Field C']" filename="test.csv"> Export </button>

    ngCsv attributes that we use here

    • ng-csv: The data array - Could be an expression, a value or a promise.
    • filename: The filename that will be stored on the user's computer.
    • csv-header: If provided, would use this attribute to create a header row.
